Fluid Dynamics ~ 30/07/2014

This project is a spin-off from the modelling work I did for my undergraduate dissertation. The application aims to represent an incompressible fluid, e.g. water.

The method I used is based on work done by Jos Stam, his origional document can be found here.

The project is written in java and open source, you can find it here, alternatively you can click on the image to download a runnable version.
